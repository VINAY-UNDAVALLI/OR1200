# OpenRISC OR1200 Processor – RTL Design and Simulation

## Overview

This project explores the **OpenRISC OR1200**, a 32-bit open-source RISC processor implemented in **Verilog HDL**. The objective is to understand the processor architecture, RTL design, pipeline operation, cache organization, memory management unit (MMU), and interrupt handling through simulation using **Icarus Verilog** and **GTKWave**.

The OR1200 is based on the OpenRISC 1000 architecture and features a 5-stage pipeline, Harvard architecture, separate instruction and data caches, MMUs, and a Wishbone bus interface.

---

## Objectives

- Study the OpenRISC OR1200 processor architecture.
- Understand the RTL implementation in Verilog HDL.
- Analyze the 5-stage instruction pipeline.
- Learn cache and MMU operation.
- Simulate the processor using Icarus Verilog.
- Verify functionality using GTKWave waveform analysis.

---

## Features

- 32-bit RISC Processor
- 5-Stage Pipeline Architecture
- Program Counter (PC)
- Register File (32 General-Purpose Registers)
- Arithmetic Logic Unit (ALU)
- Instruction Memory
- Data Memory
- Load and Store Instructions
- Basic Arithmetic and Logical Operations
- Pipeline Registers (IF, ID, EX, MEM, WB)
- Reset and Clock Control
- Verilog HDL Implementation
- Simulation using Icarus Verilog
- Waveform Analysis using GTKWave
- Modular CPU Design
- Educational RTL Implementation
